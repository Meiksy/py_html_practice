      HTML FORMS

      What are forms for? Forms are used for mant different purposes. In general though, you use forms when you want to accept inout from the user. This could be anything from a search query, to a Username and Password. HTML forms are a powerful tool for adding more functionality to your website.

      FORM TAG

      The first part of setting up a form is using the form tag. Inside the element, we'll put the different inputs for the user.


      INPUT TAG

      We use the input tag to make basic a basic inout element, but we have to use attributes to specify what type of input it should be and give the input name. Like the image tag, the input tag is an empty element. (Meaning it doesn't need a clsing tag).

      INPUT ATTRIBUTES

      Every input has to have a name attribute set, as that is what we use to identify each input when the user submits the form. Otherwise, we'd have to use each input and that input won't be submitted. We can also give each element an ID attribute.
